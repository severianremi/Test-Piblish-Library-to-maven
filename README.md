
### Requirements

- Android SDK 18+

### Gradle Dependency

##### Repository

Add repository in your root `build.gradle` file

```gradle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

##### Dependency

Add dependency to your module's `build.gradle` file:

```Groovy
dependencies {
	implementation 'com.github.severianremi:Test-Piblish-Library-to-maven:Tag'
}
```
